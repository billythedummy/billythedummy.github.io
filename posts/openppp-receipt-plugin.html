Building an Expense Tracking OpenPPP Plugin

TLDR: I built a simple OpenPPP plugin that uploads photos of receipts onto Google Drive and updates a Google Sheet with their details for the purpose of expense tracking. The plugin code is available below at the end of the article.

This is a follow-up post to the intro to OpenPPP post. Please read it first to get an overview of OpenPPP if you're unfamiliar with it.

Having hacked together the initial version of OpenPPP, it was time to dogfood it with the expense-tracking use-case mentioned in the previous post.

The requirements for the plugin are straightforward: given a photo of a receipt, it should upload it onto the Google Drive folder where I save all photos of receipts, as well as update my bookkeeping Google Sheet with the receipt's purchase item, expenditure amount, date and expenditure category.

My initial idea was to do some fancy OCR to automatically extract the receipt's total, date and maybe even infer the expenditure category. However, after trying out Tesseract.js on a few sample images via a simple test plugin, it seemed like it only worked well when the text in the image was perfectly aligned and binarized to black and white. I also ran into some issues trying to get opencv.js to work - see comments in the example tesseract example plugin, so I couldn't use it for pre-processing the image to get it into the optimal conditions.

Okay, forget OCR, let's just get the basic stuff done first. I'll just create a HTML form to allow the user (me) to fill in these details manually. Let's figure out the Google APIs next.

Setting Up Google Cloud

After reading some of Google's dev docs, I learnt that you needed a Google Cloud project to be able to use the Google Sheets and Google Drive APIs. And so I used the cloud console UI to manually enable the 2 APIs in a new project, and then create an API key that has the permission to use them.

GAPI, GSI, GIS

These stand for Google API, Google Sign-In and Google Identity Services respectively. After reading the Google js quickstart multiple times, I finally got a rough idea of how it all worked.

Basically:

  1. The GIS script sets up GIS by defining global variable window.google
  2. The GAPI script sets up GAPI by defining global variable window.gapi
  3. OAuth 2.0 access token is requested from GIS by calling window.google.accounts.oauth2.initTokenClient with the project's client ID and the required scopes (in this case, Drive + Sheets). This creates a "Sign in to Google" pop-up followed by a dialog requesting for the required scope's permissions.
  4. Use window.gapi.client to make the required API calls to Drive and Sheets, attaching the access token to requests for authentication.

Combine this with some DOM manipulation code to create the form for manual entry of expense details and that's basically the entire plugin!

Other Notes

Some other stuff to take note of about the plugin:

  • Since both Google scripts are CommonJS, I tried to just paste the required libraries in to run them, which is why you will find the entire GAPI code in the source below. However, due to the GIS script not being able to run in strict mode, it's currently loaded by way of an injected script tag.
  • Since the access token can be reused, I cache it in localStorage along with its expiry time, only re-requesting it if required.
  • GAPI needs to be initialized with the discovery docs of the Google Services you want to use.
  • If your Google Cloud project's app is not published, you will see an additional warning about it during the GIS login.

The Plugin

As promised, the entire plugin is below, ready to be copied and pasted into your OpenPPP saved plugins. You'll need to change the secrets at the top of the file to those specific to your Google Cloud and Drive configuration before use. You can also change the form generated and the Google Drive column values to suit your own format. 


/* SECRETS */
const GAPI_KEY = "<REPLACE-ME>";
const CLIENT_ID = "<REPLACE-ME>";
const SPREADSHEET_ID = "<REPLACE-ME>";
const GDRIVE_RECEIPTS_FOLDER_ID = "<REPLACE-ME>";

const SCOPES = "https://www.googleapis.com/auth/drive https://www.googleapis.com/auth/spreadsheets";

/**
  * 
  * @param {string} val 
  * @returns {HTMLOptionElement}
  */
function createCategoryOption(val) {
  const res = document.createElement("option");
  res.value = val;
  res.innerText = val;
  return res;
}

/**
  * 
  * @param {HTMLElement} elem 
  * @param {string} label
  * @returns {HTMLLabelElement}
  */
function encapsInLabel(elem, label) {
  const res = document.createElement("label");
  res.style.marginBottom = "10px";
  res.style.display = "flex";

  const span = document.createElement("span");
  span.innerText = `${label}: `;
  span.style.display = "block";

  elem.style.display = "block";
  elem.style.flex = "1 1 0%";

  res.appendChild(span);
  res.appendChild(elem);

  return res;
}

/**
  * @returns {HTMLFormElement}
  */
function createForm() {
  const form = document.createElement("form");

  form.style.display = "flex";
  form.style.flexDirection = "column";

  const date = document.createElement("input");
  date.name = "date";
  date.type = "date";
  date.value = new Date().toLocaleDateString("sv-SE");
  form.appendChild(encapsInLabel(date, "date"));

  const description = document.createElement("input");
  description.name = "description";
  description.type = "text";
  form.appendChild(encapsInLabel(description, "description"));

  const expenses = document.createElement("input");
  expenses.name = "expenses";
  expenses.type = "number";
  expenses.step = "0.01";
  form.appendChild(encapsInLabel(expenses, "expenses"));

  const category = document.createElement("select");
  category.name = "category"
  for (const val of ["Sale", "COGS", "CapEx", "Staff Welfare"]) {
    const option = createCategoryOption(val);
    category.appendChild(option);
  }
  form.appendChild(encapsInLabel(category, "category"));

  const submitBtn = document.createElement("button");
  submitBtn.type = "submit";
  submitBtn.innerText = "submit";
  form.appendChild(submitBtn);

  return form;
}

/**
  * 
  * @returns {Promise<FormData>}
  */
function createAndShowDialog() {
  const dialog = document.createElement("dialog");

  const form = createForm();
  dialog.appendChild(form);

  return new Promise((resolve, reject) => {
    const cleanupDialog = () => {
      // try catch all to prevent firing twice (once by onclose, once by click to close) from throwing errors
      try {
        dialog.close();
        document.body.removeChild(dialog);
      } catch (e) {
        console.log("dialog E: ", e);
      }
    }

    form.onsubmit = (e) => {
      e.preventDefault();
      cleanupDialog();
      resolve(new FormData(form));
    }

    const cleanupDialogReject = () => {
      cleanupDialog();
      reject(new Error("form dialog closed"));
    }

    dialog.onclose = cleanupDialogReject;
    const clickToCloseWindowListener = (event) => {
      if (event.target && event.target === dialog) {
        window.removeEventListener("click", clickToCloseWindowListener);
        cleanupDialogReject();
      }
    };

    window.addEventListener("click", clickToCloseWindowListener);
    document.body.appendChild(dialog);
    dialog.showModal();
  });
}

/**
  * 
  * @returns {Promise<any>}
  */
async function loadGapi() {
  const loadPromise = new Promise((resolve) => {
    // @ts-ignore
    window.gapi.load("client", resolve);
  });
  await loadPromise;
  // @ts-ignore
  const client = window.gapi.client;
  await client.init({
    apiKey: GAPI_KEY,
    discoveryDocs: ["https://www.googleapis.com/discovery/v1/apis/drive/v3/rest", "https://sheets.googleapis.com/$discovery/rest?version=v4"],
  });
  return client;
}

/**
  * 
  * @param {number} ms 
  * @returns {Promise<void>}
  */
function sleep(ms) {
  return new Promise(resolve => { setTimeout(resolve, ms); });
}

/**
  * https://accounts.google.com/gsi/client is a pile of cjs
  * that can't run in strict mode due to some undefined variables
  * @returns {Promise<void>}
  */
function loadGis() {
  // @ts-ignore
  if (window.google) {
    return Promise.resolve();
  }
  const script = document.createElement("script");
  script.src = "https://accounts.google.com/gsi/client";
  script.id = "tempgsiscript";
  document.body.appendChild(script);

  return new Promise(async (resolve, reject) => {
    for (let i = 0; i < 100; i++) {
      // @ts-ignore
      if (window.google) {
        document.body.removeChild(script);
        resolve();
        return;
      }
      await sleep(100);
    }
    reject("timeout waiting for google");
  })
}

/**
  * @returns {Promise<string>}
  */
function loadToken() {
  const expiryTimeLocalStorageKey = "gisexpirytime";
  const localStorageKey = "gisaccesstoken";
  const savedExpiry = window.localStorage.getItem(expiryTimeLocalStorageKey);
  if (savedExpiry) {
    try {
      const expiryTimestamp = Number.parseInt(savedExpiry);
      if (Date.now() < expiryTimestamp) {
        const saved = window.localStorage.getItem(localStorageKey);
        if (!saved) {
          throw new Error("expiry exists but saved doesnt");
        }
        return Promise.resolve(saved);
      }
    } catch (e) {
      console.error("saved", e);
    }
  }
  // saved token doesnt exist or expired, re-request
  window.localStorage.removeItem(expiryTimeLocalStorageKey);
  window.localStorage.removeItem(localStorageKey);
  return new Promise((resolve, reject) => {
    // @ts-ignore
    const tokenClient = window.google.accounts.oauth2.initTokenClient({
      client_id: CLIENT_ID,
      scope: SCOPES,
      callback: ({ access_token, expires_in }) => {
        window.localStorage.setItem(expiryTimeLocalStorageKey, (Date.now() + expires_in * 1000).toString());
        window.localStorage.setItem(localStorageKey, access_token);
        resolve(access_token);
      },
      error_callback: (e) => {
        console.error("GIS", e);
        reject(e);
      },
    });
    tokenClient.requestAccessToken();
  });
}

/**
  * 
  * @param {any} client 
  * @param {FormData} formData 
  */
async function writeGSheets(client, formData) {
  const { result: { values } } = await client.sheets.spreadsheets.values.get({
    spreadsheetId: SPREADSHEET_ID,
    range: "Main!B:B"
  });
  let ct = 0;
  while (values[ct] && values[ct][0] != "") {
    ct++;
  }
  const row = ct + 1;
  // @ts-ignore
  const day = new Date(formData.get("date")).getDate();
  const description = formData.get("description");
  const expenses = formData.get("expenses");
  const category = formData.get("category");
  const toWrite = [[day, description, "", expenses, category]];

  await client.sheets.spreadsheets.values.update({
    spreadsheetId: SPREADSHEET_ID,
    range: `Main!B${row}:F${row}`,
    valueInputOption: "USER_ENTERED",
    resource: {
      values: toWrite,
    }
  });
}

/**
  * 
  * @param {any} client
  * @param {File} file 
  * @param {string} fileName 
  */
async function uploadReceiptImage(client, file, fileName) {
  const fileMetadata = {
    name: fileName,
    mimeType: file.type,
    parents: [GDRIVE_RECEIPTS_FOLDER_ID],
  };
  const form = new FormData();
  form.append("metadata", new Blob([JSON.stringify(fileMetadata)], { type: 'application/json' }));
  form.append("file", file);

  const token = client.getToken().access_token;
  const res = await fetch(
    "https://www.googleapis.com/upload/drive/v3/files?uploadType=multipart&supportsAllDrives=true",
    {
      method: "POST",
      headers: new Headers({ Authorization: `Bearer ${token}` }),
      body: form,
    }
  );
  if (!res.ok) {
    const errTxt = await res.text();
    throw new Error(errTxt);
  }
}

/** @type {OpenPPPHandler} */
async function main(image) {
  const [formData, client] = await Promise.all([
    createAndShowDialog(),
    loadGapi(),
    loadGis(),
  ]);
  const token = await loadToken();
  client.setToken({ access_token: token });

  const imgName = `${formData.get("date")} ${formData.get("description")}`;
  await Promise.all([
    uploadReceiptImage(client, image, imgName),
    writeGSheets(client, formData),
  ]);
}
